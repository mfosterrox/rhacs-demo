= RHACS Demo Environment Setup

== Getting Started

Clone the repository:

[source,bash]
----
if [ -d rhacs-demo ]; then cd rhacs-demo && git pull; else git clone https://github.com/mfosterrox/rhacs-demo.git && cd rhacs-demo; fi && ./install.sh
----

== Prerequisites

Before running the installation script, ensure the following variables are set in your `~/.bashrc`:

* `ROX_CENTRAL_URL` - RHACS Central URL for API access and roxctl CLI (e.g., https://central.example.com)
* `ROX_PASSWORD` - RHACS Central password for API access and roxctl CLI

Optional variables:

* `RHACS_NAMESPACE` - Namespace where RHACS is installed (default: `stackrox`)
* `RHACS_ROUTE_NAME` - Name of the RHACS route (default: `central`)
* `RHACS_VERSION` - Optional: Target RHACS version

== What It Does

The installation script will:

. Verify all required variables and credentials are configured
. Verify RHACS installation including secured cluster services
. Ensure the RHACS route is encrypted with TLS
. Check and update RHACS to the latest stable version if needed
